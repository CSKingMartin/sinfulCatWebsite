Hello world! Welcome to SinfulCat
SinfulCat is the brainchild of Conner Wells, a Computer Science student at Linfield College. This framework was desgined to help students and other Web Developers get a foot in the door using useful tools to help streamline workflow.
Ready to start learning tools like Gulp, Handlbars, and SCSS? SinfulCat is a framework built from Gulp designed to be as informative as possible, allowing you to have confidence in your technique and allow you to work faster. This means you get to spend less time honing your HTML and CSS skills, and less time with frustrating snags in the development process.
For more information about the project, please visit the about page, or visit the Github Repository .

About

Gulp

Gulp is the streaming build system that uses Javascript in your development space to automate your workflow. Gulp is used by SinfulCat to do things like host your webpages on your browser, refresh them when changed, and precompile code for distribution.

Learn more about Gulp!
Handlebars.js

Handlebars.js is a templating language library that allows you use logic with HTML. When writing Handlebars files, you can reuse HTML classes within your project, and populate the data akin to functional programming.

Learn more about Handlebars!
Sass/SCSS

Sass or SCSS are both similar options to help write cleaner, more organized CSS. Using this language library allows users to separate CSS into easily readable chunks which allows for clearer organization during development.

Learn more about Sass/SCSS!

Theory

Core Skills

SinfulCat was designed with you in mind, because it was designed for everyone! Step by step instruction combined with several included examples are here to help get you confident in your abilities as a developer.

No Magic Behind the Curtain

The tools chosen for SinfulCat were carefully selected so that everything is out in the open. Learning with other high-end build-tools like Ruby on Rails have too much of the process hidden in the code. With SinfulCat, the key parts of the framework are given to you piece by piece so you know exactly what's going on all the time.

A Foot in the Door

The Web Development industry has an extremely high turnover rate with development tools. If the tools are changing all the time, how is someone outside of the industry supposed to keep up? The practices employed in SinfulCat are focused on overarching skills with HTML and CSS that will help you anywhere that are independent on the tools you use.
